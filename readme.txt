== Description ==

Add Hreflang meta tags to the head of your Multi-Language WordPress Website. It is compatible with the elementor plugin.

== Installation ==

Unzip plugin files and upload them under your '/wp-content/plugins/' directory.

Resulted names will be: './wp-content/plugins/hoo-hreflang-tags/*'

Activate plugin at "Plugins" administration page.
